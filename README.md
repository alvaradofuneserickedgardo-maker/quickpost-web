# quickpost-web

**Página web de ventas QuickPost IA**

QuickPost IA es una plataforma que te permite automatizar las ventas y la gestión de publicaciones en redes sociales utilizando inteligencia artificial. Integra herramientas como Hotmart, Telegram, Mailchimp y Make, facilitando la creación de flujos multi-escenario para atraer tráfico y gestionar ofertas de manera eficiente. El proyecto está pensado para emprendedores y negocios digitales que buscan simplificar y potenciar sus procesos de ventas online.

### Características principales
- Automatización de publicaciones y ventas en redes sociales.
- Integración con sistemas de pago (Hotmart).
- Flujo multi-escenario con Google Sheets, Telegram y YouTube.
- Seguimiento y gestión de ofertas.
- Conexión sencilla con otras plataformas mediante Make y Zapier.
